# Go-Live Gate Checklist

## Catalog readiness
- [ ] 20+ listings ready with title, description, 13 tags
- [ ] All product files downloadable and correctly packaged
- [ ] Thumbnail/gallery assets exported in required dimensions

## Channel readiness
- [ ] Etsy shop branding complete (banner, logo, about, policies)
- [ ] Shopify homepage + product pages + legal pages live
- [ ] Gumroad bundles published and linked

## Checkout and fulfillment
- [ ] Checkout test succeeds on each active channel
- [ ] Digital files auto-deliver after purchase
- [ ] POD routing tested (Printify integration)

## Analytics and operations
- [ ] KPI tracker receives daily updates
- [ ] Daily and weekly cron jobs active
- [ ] Support inbox monitored and template replies prepared

## Final release criteria
- [ ] No P0 issues in QA
- [ ] Mobile purchase flow validated
- [ ] Domain and SEO metadata configured
