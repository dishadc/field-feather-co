# SOP: Etsy SEO Optimization Loop (Weekly)

## Inputs
- Etsy stats export (views, favorites, orders)
- Current listing titles/tags
- Competitor snapshots (top 10 for target keyword)

## Weekly loop
1. Rank listings by visits with low conversion.
2. For top 5 underperformers:
   - Rewrite first 40 title characters
   - Replace 3 lowest-intent tags with long-tail tags
   - Improve first image contrast/clarity
3. For top 3 performers:
   - Duplicate concept into adjacent long-tail variants
4. Log changes in `data/seo_change_log.csv`

## KPI target
- Improve listing conversion by +0.3pp over 2 weeks.
