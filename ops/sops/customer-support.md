# SOP: Customer Support (Digital Products)

## SLA
- First response within 24 hours.
- Resolution target within 48 hours.

## Triage categories
1. Download not received
2. File corrupted/unopenable
3. Wrong product purchased
4. Refund request
5. General usage question

## Response templates
- Download issue: confirm order email, resend direct download instructions, provide alternate file link.
- Corrupted file: send fresh file package and checksum/size details.
- Wrong purchase: offer exchange or refund per policy.

## Refund policy default
- Digital goods are generally non-returnable after download unless file defect exists.
- Goodwill one-time refund allowed for first-time buyer friction.

## Escalation
- 2+ unresolved back-and-forths -> owner escalation same day.
