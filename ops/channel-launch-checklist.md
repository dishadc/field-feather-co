# Channel Launch Checklist

Purpose: final launch checklist for each commerce channel.

## Shared go-live requirements
- [ ] Brand name, logo treatment, and support email consistent
- [ ] Terms, privacy, and refund language published or linked
- [ ] Product titles, descriptions, pricing, and thumbnail set approved
- [ ] Download files uploaded and opened successfully after upload
- [ ] Payout / payment method verified
- [ ] Test order path defined
- [ ] Support response workflow documented

## Etsy
- [ ] Seller account active and in good standing
- [ ] Shop banner, icon, announcement, about section, FAQ, and policies complete
- [ ] Digital listing files uploaded to each listing
- [ ] 13 tags added per listing
- [ ] Opening promo/coupon configured
- [ ] Shipping profile not incorrectly applied to digital goods
- [ ] Shop language and return policy consistent with digital products
- [ ] First 15 launch listings published
- [ ] Test purchase completed or equivalent delivery verification performed

## Gumroad
- [ ] Account verified
- [ ] Payout method connected and accepted
- [ ] Product cover images uploaded
- [ ] Product descriptions include personal-use licensing
- [ ] Product files attached and downloadable
- [ ] Product enabled for sale
- [ ] Checkout URLs captured into `data/product_links.csv`
- [ ] Bundle cross-sells configured
- [ ] Test purchase completed

## Shopify
- [ ] Store admin access confirmed
- [ ] Theme customized to brand system
- [ ] Domain / subdomain connected if used
- [ ] Product pages loaded with images, descriptions, and files
- [ ] Digital delivery app configured
- [ ] Payments configured
- [ ] Tax settings reviewed
- [ ] Contact, terms, privacy, and refund pages published
- [ ] Navigation and footer links complete
- [ ] Test checkout completed

## Pinterest
- [ ] Business profile complete
- [ ] Bio, profile image, and website linked
- [ ] Initial boards created for warblers, state checklists, migration, planners
- [ ] 15–30 launch pins scheduled or published
- [ ] UTM or destination links verified
- [ ] Rich pin / domain claim completed if used

## Launch evidence to save
- [ ] Screenshot of live product page on each active channel
- [ ] Screenshot of successful checkout or order confirmation
- [ ] Order/receipt record saved
- [ ] Updated readiness report archived
- [ ] KPI tracker updated with launch date

## Current blockers snapshot
- Etsy blocked by approval / access gate
- Gumroad publishing blocked until payout method is connected
- Shopify blocked by anti-bot verification gate
