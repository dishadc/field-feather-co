# Launch 20 SKU Media Shotlist

For each SKU publish with:
- 1 cover image (high contrast title + product format)
- 3 interior page previews
- 1 features graphic (what's included)
- 1 use-case mockup (tablet or printed sheet)
- 1 FAQ panel (instant download, no physical item)
- 1 optional 5–10 second listing video

## Thumbnail copy formula
"{State/Theme} Birding Checklist" + "Instant Download" + price anchor.

## Listing image sequence
1. Cover
2. Included pages overview
3. Close-up readability sample
4. How to use
5. Device + print mockup
6. FAQ / terms
